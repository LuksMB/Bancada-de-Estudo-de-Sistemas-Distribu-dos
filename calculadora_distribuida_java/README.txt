A calculadora funciona com o cliente enviando as informações no formato "operação,numero1,numero2", e logo em seguida "localhost" (ou qualquer que seja o IP do server) por meio de argumentos na linha de comando.

Os seguintes argumentos são aceitos como operação:

    "add" - Adição
    "sub" - Subtração
    "div" - Divisão
    "mult" - Multiplicação

Por ser feito utilizando linhas de comando, as conexões e o programa é encerrado logo que receba a resposta da calculadora, sendo assim, não possui uma interface interativa.

A intenção do programa é simples e ainda não está completamente livre de erros.
